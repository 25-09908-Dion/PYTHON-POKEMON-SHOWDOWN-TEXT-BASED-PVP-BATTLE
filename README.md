# PYTHON-POKEMON-SHOWDOWN-TEXT-BASED-PVP-BATTLE

Inspired by the Pokémon series and the team’s childhood experiences, this project combines nostalgia with programming skills. By creating a text-based Pokémon-inspired battle game, the team applied fundamental programming concepts while developing a project that reflects both their personal interests and technical growth.

The game also incorporates a Filipino local twist through its original story and backstory, adding a unique cultural element to the gameplay experience. Players can further explore the game by completing Story Mode, which unlocks additional features and content as a reward for their progress.
